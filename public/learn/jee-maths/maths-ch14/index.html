Probability

This chapter of JEE Maths covers 17 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Define an event as a subset of the sample space — Recognise that any subset of the sample space of a random experiment is an event, and translate a described happening into the matching subset of S.
  2. Decide whether an event has occurred from the outcome — Use the rule that an event E occurs exactly when the actual outcome of the experiment belongs to E, and does not occur when the outcome lies outside E.
  3. Identify impossible and sure events — Recognise the empty set as the impossible event and the whole sample space as the sure event, and classify a described event as one of these when appropriate.
  4. Identify a simple (elementary) event — Recognise an event with exactly one sample point as a simple event, and count that a space of n outcomes has exactly n simple events.
  5. Identify a compound event — Recognise an event with more than one sample point as a compound event, and write the subset of S it corresponds to.
  6. Form the complementary event 'not A' — Construct the complement A' of an event A as the set of outcomes of S not in A, i.e. A' = S - A, the event 'not A'.
  7. Form the event 'A or B' as the union — Build the event 'A or B' as the union A union B, containing every outcome in A, in B, or in both.
  8. Form the event 'A and B' as the intersection — Build the event 'A and B' as the intersection A intersect B, containing exactly the outcomes common to both events.
  9. Form the event 'A but not B' as the difference — Build the event 'A but not B' as the set difference A - B, equal to A intersect B', the outcomes in A that are excluded from B.
  10. Test whether two events are mutually exclusive — Decide that two events are mutually exclusive exactly when they cannot occur together, i.e. when their intersection is empty (disjoint sets).
  11. Test whether events are exhaustive (and mutually exclusive) — Decide that events are exhaustive when their union is the whole sample space, and mutually exclusive and exhaustive when they are also pairwise disjoint.
  12. State the axioms of probability — State the axiomatic definition of probability as a function on the power set of S into [0,1] satisfying non-negativity, P(S)=1, and additivity for mutually exclusive events.
  13. Compute P(A) by summing the probabilities of its outcomes — Find the probability of any event by adding the assigned probabilities of the sample points it contains: P(A) = sum of P(omega_i) over omega_i in A.
  14. Compute probability for equally likely outcomes as n(A)/n(S) — When all outcomes are equally likely, compute the probability of an event as the ratio of favourable outcomes to total outcomes, P(A) = n(A)/n(S).
  15. Apply the addition theorem P(A or B) = P(A) + P(B) - P(A and B) — Compute the probability of a union with the addition theorem, subtracting the overlap, and reduce it to P(A) + P(B) when the events are mutually exclusive.
  16. Apply the complement rule P(not A) = 1 - P(A) — Use the complement rule to find the probability of 'not A' from P(A), exploiting that A and A' are mutually exclusive and exhaustive.
  17. Solve probability problems using events and the axiomatic rules — Combine event algebra with the axiomatic rules — equally likely counting, the addition theorem and the complement rule — to compute the probability of any event in a random experiment.
