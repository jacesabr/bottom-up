Linear Inequalities

This chapter of JEE Maths covers 11 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Recognise an inequality and the four order symbols — Identify a statement as an inequality when two numbers or expressions are related by one of the symbols <, >, less-than-or-equal, or greater-than-or-equal, distinguishing it from an equation.
  2. Define a solution and the solution set of an inequality — State that a solution is any value of the variable that makes the inequality a true statement, and that the set of all such values is the solution set.
  3. Apply the two rules for manipulating an inequality (reverse the sign on a negative) — Add or subtract the same number on both sides freely, and multiply or divide both sides by a number - reversing the inequality sign whenever that number is negative.
  4. Classify inequalities as numerical/literal/double and as strict or slack — Sort inequalities into numerical, literal, and double, and separately into strict (<, >) versus slack (less-or-equal, greater-or-equal).
  5. Identify a linear inequality in one or two variables — Recognise ax + b (order) 0 as linear in one variable and ax + by (order) c as linear in two, and tell these apart from quadratic inequalities.
  6. Solve a linear inequality in one variable algebraically — Use the two rules to isolate the variable in a linear inequality and write the solution set over the natural numbers, integers, or real numbers.
  7. Represent the solution of an inequality on a number line — Graph a one-variable inequality's solution on a number line, using an open circle for strict and a dark circle for inclusive endpoints, with the line shaded left or right.
  8. Translate a word problem into a linear inequality and solve it — Set up a linear inequality from a statement problem (averages, ranges, mixtures), solve it, and interpret the result in context.
  9. Solve a double inequality by operating on all three parts — Solve an inequality of the form p (order) expression (order) q by applying the rules to all three parts at once, reversing both signs on a negative multiplier.
  10. Solve a system of linear inequalities and take the common solution — Solve each inequality of a system separately, then find the values common to all of them, reading off the overlap on a number line.
  11. Solve and apply linear inequalities in one variable — Bring the rules, algebraic solving, number-line representation, word-problem modelling, double inequalities, and systems together to solve and apply any linear inequality in one variable.
