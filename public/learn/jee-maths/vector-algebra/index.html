Vector Algebra

This chapter of JEE Maths covers 13 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Define a vector — Recognise a vector as a quantity having both magnitude and direction.
  2. Find the magnitude (length) of a vector — Define the magnitude as the distance between the initial and terminal points.
  3. Define the position vector of a point — Recognise the position vector of P as the vector OP from the origin to P.
  4. Find the direction cosines of a vector — Identify the direction cosines l, m, n as the cosines of the angles with the axes.
  5. Recognise the zero (null) vector — Identify the zero vector as one whose initial and terminal points coincide.
  6. Find the unit vector in a direction — Recognise a unit vector as one of magnitude 1, and form it as a / |a|.
  7. Recognise collinear vectors — Identify collinear vectors as those parallel to the same line.
  8. Recognise equal vectors — Identify equal vectors as those with the same magnitude AND direction, regardless of position.
  9. Add vectors by the triangle and parallelogram laws — Add two vectors head-to-tail (triangle law) or as adjacent sides of a parallelogram.
  10. Write a vector in component form — Express a vector as x i + y j + z k with scalar components x, y, z.
  11. Apply the section formula for a dividing point — Find the position vector of a point dividing a segment in a given ratio, internally or externally.
  12. Compute the scalar (dot) product — Use a.b = |a||b|cos(theta); equivalently a1 b1 + a2 b2 + a3 b3.
  13. Compute the vector (cross) product — Use a x b = |a||b| sin(theta) n-hat; it is a vector perpendicular to both, anticommutative.
