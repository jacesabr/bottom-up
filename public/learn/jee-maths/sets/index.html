Sets

This chapter of JEE Maths covers 19 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Decide whether a collection is a well-defined set — Test a collection against the well-defined criterion: a set is a collection for which membership of any object can be decided unambiguously.
  2. Use the membership symbols belongs-to and does-not-belong-to — Read and write 'a belongs to A' and 'b does not belong to A' using the symbols ∈ and ∉.
  3. Represent a set in roster (tabular) form — List all the elements of a set inside braces, separated by commas, taking each distinct element once and ignoring order.
  4. Represent a set in set-builder form — Describe a set by the common defining property of its elements, written as {x : x has property P}.
  5. Identify the empty (null) set — Recognise a set that contains no element at all, denoted by ∅ or { }.
  6. Classify a set as finite or infinite — Decide whether a set has a definite (countable) number of elements (finite) or not (infinite), using n(S) for the number of elements.
  7. Decide whether two sets are equal — Two sets are equal exactly when they have the same elements; repetition and order do not matter.
  8. Count all subsets using the power-set formula — A set with n elements has exactly 2^n subsets in total; the collection of all those subsets is called the power set P(A).
  9. Determine subset and proper-subset relations — Test whether A is a subset of B (every element of A is in B), and recognise proper subsets, supersets, and the conventions A subset A and phi subset every set.
  10. Name the standard number-set subsets of the reals — Identify N, Z, Q, T as subsets of R and state the containment relations among them.
  11. Convert between intervals and set-builder subsets of R — Translate open, closed and half-open intervals to and from set-builder form, and find an interval's length.
  12. Choose a universal set for a given context — Identify a basic set U that contains all the sets relevant to a context, so every set under study is a subset of U.
  13. Represent sets and their relations with Venn diagrams — Draw the universal set as a rectangle and its subsets as circles, placing elements and showing subset relations.
  14. Compute the union of two sets — Form A union B as the set of all elements in A or in B (common elements taken once), and use its basic properties.
  15. Compute the intersection of two sets — Form A intersection B as the elements common to both A and B, recognise disjoint sets, and use intersection's properties including distributivity.
  16. Compute the difference of two sets — Form A - B as the elements in A but not in B, and note that A - B is generally not equal to B - A.
  17. Find the complement of a set relative to U — Form A' = U - A, the elements of the universal set that are not in A, and use the double-complement law (A')' = A.
  18. Apply De Morgan's laws to complements of union and intersection — Use (A union B)' = A' intersection B' and (A intersection B)' = A' union B' to simplify complements of combined sets.
  19. Represent sets and solve problems using set operations — Combine set representation, subset and equality reasoning, and the operations of union, intersection, difference and complement (with De Morgan's laws) to solve problems about collections.
