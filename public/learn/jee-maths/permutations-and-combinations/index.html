Permutations and Combinations

This chapter of JEE Maths covers 12 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Apply the fundamental principle of counting (multiplication principle) — Count the ways a sequence of independent choices can be made by multiplying the number of ways for each choice.
  2. Recognise a permutation as an ordered arrangement — Identify when a counting problem is a permutation: the objects are arranged in a definite order, so the order of writing them matters.
  3. Evaluate and simplify factorial expressions — Use n! as the product of the first n natural numbers, the convention 0! = 1, and the recursion n! = n(n-1)! to evaluate and cancel factorials.
  4. Count permutations of distinct objects with nPr = n!/(n-r)! — Use the formula nPr = n!/(n-r)! for the number of ordered arrangements of n distinct objects taken r at a time (no repetition).
  5. Count permutations when repetition is allowed (n^r) — Use n^r for the number of ordered arrangements of n distinct objects taken r at a time when each object may be reused.
  6. Count permutations when some objects are identical (n!/(p1!p2!...pk!)) — Divide n! by the factorials of the repeat counts to count arrangements of n objects in which some objects are indistinguishable.
  7. Recognise a combination as an unordered selection — Identify when a counting problem is a combination: objects are selected without regard to order, so AB and BA are the same.
  8. Relate permutations and combinations by nPr = nCr x r! — Use the fact that each combination of r objects yields r! permutations to connect nPr and nCr.
  9. Count combinations with nCr = n!/(r!(n-r)!) — Use nCr = n!/(r!(n-r)!) to count the number of unordered selections of r objects from n, including nC0 = nCn = 1.
  10. Use the symmetry nCr = nC(n-r) — Apply nC(n-r) = nCr and the consequence that nCa = nCb implies a = b or a + b = n.
  11. Apply Pascal's rule nCr + nC(r-1) = (n+1)Cr — Use the identity nCr + nC(r-1) = (n+1)Cr that combines two adjacent combinations into one of the next row.
  12. Solve counting problems by choosing arrangement vs selection and combining the rules — Model any counting problem by deciding whether order matters (permutation) or not (combination), then combine nPr, n^r, n!/(p1!...), nCr with the multiplication and addition principles.
