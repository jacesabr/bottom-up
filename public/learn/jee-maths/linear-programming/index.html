Linear Programming

This chapter of JEE Maths covers 15 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Recognise an optimisation problem — Identify an optimisation problem as one that seeks to maximise or minimise a quantity such as profit or cost.
  2. Write the objective function — Form the linear objective function Z = ax + by that must be maximised or minimised.
  3. Identify the decision variables — Recognise the variables x and y in the objective function as the decision variables.
  4. Write the linear constraints — Express the restrictions on the variables as linear inequalities called constraints.
  5. Apply the non-negative restrictions — Recognise the conditions x >= 0, y >= 0 as the non-negative restrictions on the decision variables.
  6. Define a linear programming problem — State an LPP as finding the optimal value of a linear objective function subject to linear constraints and non-negativity.
  7. Identify the feasible region — Recognise the feasible region as the common region determined by all the constraints including non-negativity.
  8. Recognise feasible and infeasible solutions — Identify points within and on the boundary of the feasible region as feasible solutions, others as infeasible.
  9. Recognise an optimal solution — Identify an optimal solution as a feasible point that gives the optimal (max or min) value of the objective function.
  10. Decide if a feasible region is bounded or unbounded — Classify a feasible region as bounded if it can be enclosed within a circle, otherwise unbounded.
  11. Find the corner points of the feasible region — Identify corner points (vertices) as intersections of two boundary lines of the feasible region.
  12. Apply Theorem 1: optimum at a corner point — Use Theorem 1: when Z has an optimal value it must occur at a corner point of the feasible region.
  13. Apply Theorem 2: bounded region has both max and min — Use Theorem 2: if R is bounded, Z has both a maximum and a minimum, each at a corner point.
  14. Recognise multiple optimal solutions — Detect that when two corner points give the same optimal value, every point on the segment joining them is also optimal.
  15. Solve an LPP by the Corner Point Method — Solve a linear programming problem graphically by evaluating Z at the corner points of the feasible region.
