Matrices

This chapter of JEE Maths covers 16 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Define a matrix as a rectangular array — Recognise a matrix as an ordered rectangular array of numbers or functions, whose entries are its elements.
  2. State the order of a matrix — Describe a matrix with m rows and n columns as an m x n matrix, with mn elements.
  3. Recognise a square matrix — Identify a square matrix as one with equal numbers of rows and columns (m = n).
  4. Recognise a diagonal matrix — Identify a diagonal matrix as a square matrix whose non-diagonal elements are all zero.
  5. Recognise a scalar matrix — Identify a scalar matrix as a diagonal matrix whose diagonal elements are all equal.
  6. Recognise the identity matrix — Identify the identity matrix as a square matrix with 1s on the diagonal and 0s elsewhere.
  7. Recognise the zero (null) matrix — Identify the zero matrix as one in which all elements are zero.
  8. Test two matrices for equality — Two matrices are equal iff they have the same order and equal corresponding elements.
  9. Add two matrices of the same order — Add matrices entrywise; the operation is defined only for matrices of the same order.
  10. Multiply a matrix by a scalar — Multiply a matrix by a scalar k by multiplying every element by k.
  11. Multiply two matrices — Multiply A by B when columns of A = rows of B, taking row-by-column dot products.
  12. Recognise that matrix multiplication is not commutative — Understand that AB need not equal BA, and AB may be defined while BA is not.
  13. Form the transpose of a matrix — Form A' (or A^T) by interchanging the rows and columns of A.
  14. Recognise a symmetric matrix — Identify a symmetric matrix as a square matrix with A' = A.
  15. Recognise a skew-symmetric matrix — Identify a skew-symmetric matrix as a square matrix with A' = -A and zero diagonal.
  16. Define an invertible matrix and its inverse — Recognise an invertible square matrix A as one with a square B of the same order satisfying AB = BA = I; B = A-inverse is unique.
