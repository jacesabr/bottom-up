Inverse Trigonometric Functions

This chapter of JEE Maths covers 13 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Recall that an inverse exists only for a one-one and onto function — Recall from Chapter 1 that f has an inverse only if it is one-one and onto; trig functions are not, over their natural domains.
  2. Define the inverse g of a bijective function f — For a one-one onto f : X -> Y, define the unique inverse g : Y -> X with domain of g = range of f and range of g = domain of f.
  3. Understand a branch of an inverse trig function — Recognise that each interval over which a trig function is bijective gives a different branch of its inverse.
  4. State the principal value branch of sin-inverse — Take sin-inverse with domain [-1, 1] and principal range [-pi/2, pi/2].
  5. State the principal value branch of cos-inverse — Take cos-inverse with domain [-1, 1] and principal range [0, pi].
  6. State the principal value branch of tan-inverse — Take tan-inverse with domain R and principal range (-pi/2, pi/2).
  7. State the principal value branch of cot-inverse — Take cot-inverse with domain R and principal range (0, pi).
  8. State the principal value branch of cosec-inverse — Take cosec-inverse with domain R - (-1, 1) and principal range [-pi/2, pi/2] minus {0}.
  9. State the principal value branch of sec-inverse — Take sec-inverse with domain R - (-1, 1) and principal range [0, pi] minus {pi/2}.
  10. Distinguish sin-inverse x from (sin x) to the power -1 — Avoid the classic trap: sin-inverse x is an angle, NOT the reciprocal 1/sin x.
  11. Define the principal value of an inverse trig function — Define the principal value as the value of the inverse function that lies in the principal branch range.
  12. Apply sin(sin-inverse x) = x and sin-inverse(sin x) = x on the right domains — Use the cancellation identities, valid only when x lies in the correct domain.
  13. Evaluate principal values of inverse trig expressions — Combine the principal-branch ranges and cancellation identities to evaluate inverse trig expressions correctly.
