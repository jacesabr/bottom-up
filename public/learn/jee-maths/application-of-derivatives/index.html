Application of Derivatives

This chapter of JEE Maths covers 12 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Interpret the derivative as a rate of change — Read dy/dx as the rate of change of y with respect to x, and chain rates through a common variable.
  2. Define increasing and decreasing on an interval — Define f increasing on I if x1 < x2 implies f(x1) <= f(x2), decreasing if f(x1) >= f(x2).
  3. Use the sign of f'(x) to test monotonicity — Apply: f is increasing where f'(x) >= 0 and decreasing where f'(x) <= 0.
  4. Define maximum and minimum value on an interval — f has a maximum value at c if f(c) > f(x) for all x in I; a minimum if f(c) < f(x).
  5. Define local maxima and minima — Recognise a local maximum/minimum at an interior point c where f(c) is greatest/least nearby.
  6. Use that a local extremum forces f'(c) = 0 or non-differentiability — Apply Theorem 2: at a local maxima/minima, either f'(c) = 0 or f is not differentiable at c.
  7. Identify the critical points of a function — Find critical points: where f'(x) = 0 or f is not differentiable.
  8. Classify critical points by the first derivative test — Use the sign change of f' across a critical point to decide maxima, minima or inflexion.
  9. Classify critical points by the second derivative test — At f'(c) = 0, use the sign of f''(c) to decide local maxima (<0) or minima (>0).
  10. Define absolute (global) maximum and minimum — Recognise the absolute maximum/minimum as the greatest/least value of f over the whole interval.
  11. Use that a continuous function on [a,b] attains its extrema — Apply Theorem 5: a continuous function on a closed interval attains both its absolute maximum and minimum.
  12. Find absolute extrema of a function on a closed interval — Apply the working rule: evaluate f at all critical points and at the end points, then pick the greatest and least.
