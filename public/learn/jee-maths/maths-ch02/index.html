Relations and Functions

This chapter of JEE Maths covers 14 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Identify an ordered pair and when two ordered pairs are equal — Recognise an ordered pair (p, q) as two elements written in a fixed order, and apply the rule that two ordered pairs are equal only when both corresponding entries match.
  2. Form the cartesian product A x B of two sets — Construct P x Q as the set of all ordered pairs (p, q) with p from P and q from Q, and handle the empty-set case.
  3. Count the elements of a cartesian product as n(A) x n(B) — Use the rule that if A has p elements and B has q elements then A x B has pq elements, and apply it to count pairs.
  4. Define a relation R from A to B as a subset of A x B — Recognise a relation as a subset of the cartesian product picked out by a relationship between first and second elements, and write it in roster or set-builder form.
  5. Find the domain, range and codomain of a relation — Read off the domain (all first elements), the range (all second elements) and the codomain (the whole second set) of a relation, noting range is a subset of codomain.
  6. Count the number of relations from A to B as 2^(pq) — Use the fact that a relation is any subset of A x B to count the relations from A to B as 2 raised to the power n(A x B) = pq.
  7. Decide whether a relation is a function — Apply the test that a relation is a function when every element of the domain set A has one and only one image, so no two distinct pairs share a first element.
  8. Identify a real valued function and a real function — Classify a function as real valued (range in R) and as a real function (domain and range both in R), and evaluate it at given inputs.
  9. Recognise and graph the identity function — Identify f(x) = x as the identity function, with domain and range R and a straight-line graph through the origin.
  10. Recognise and graph the constant function — Identify f(x) = c as the constant function, with domain R and range the single value {c}, graphed as a line parallel to the x-axis.
  11. Identify a polynomial function — Recognise f(x) = a0 + a1 x + ... + an x^n with non-negative integer powers as a polynomial function, and tell polynomial from non-polynomial expressions.
  12. Identify a rational function and its domain — Recognise a rational function as a ratio f(x)/g(x) of polynomials defined where g(x) is not zero, and exclude the zeros of the denominator from its domain.
  13. Combine real functions by sum, difference, product, scalar and quotient — Form new real functions pointwise: (f+g), (f-g), (fg), a scalar multiple (alpha f) and the quotient (f/g), the last only where g is non-zero.
  14. Link sets by relations and decide which relations qualify as functions — Pair elements of two sets via the cartesian product, carve out a relation, read its domain, range and codomain, and decide whether it qualifies as a function.
