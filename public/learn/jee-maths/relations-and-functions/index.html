Relations and Functions

This chapter of JEE Maths covers 14 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Define a relation as a subset of A x B — Treat a relation R from A to B as an arbitrary subset of the Cartesian product A x B, dropping the everyday idea of a 'recognisable link'.
  2. Recognise the empty relation — Identify the empty relation: the relation containing no pairs, where no element of A is related to any element of A.
  3. Recognise the universal relation — Identify the universal relation: the relation A x A in which every element of A is related to every element of A.
  4. Test a relation for reflexivity — Decide whether a relation is reflexive: every element must be related to itself, (a, a) in R for every a in A.
  5. Test a relation for symmetry — Decide whether a relation is symmetric: whenever (a1, a2) is in R, (a2, a1) must also be in R.
  6. Test a relation for transitivity — Decide whether a relation is transitive: if (a1, a2) and (a2, a3) are in R, then (a1, a3) must be in R.
  7. Define and prove an equivalence relation — Recognise an equivalence relation as one that is reflexive, symmetric AND transitive, and prove it by checking all three.
  8. Form equivalence classes and partitions — Understand how an equivalence relation splits a set into disjoint equivalence classes whose union is the whole set.
  9. Test a function for one-one (injective) — Decide whether a function is one-one: distinct inputs give distinct outputs, i.e. f(x1) = f(x2) implies x1 = x2.
  10. Test a function for onto (surjective) — Decide whether a function is onto: every element of the co-domain Y is the image of some element of X.
  11. Recognise a bijective (one-one and onto) function — Identify a bijection as a function that is both one-one and onto.
  12. Compute the composition of two functions — Form the composite gof of f : A -> B and g : B -> C by (gof)(x) = g(f(x)).
  13. Define an invertible function and its inverse — Recognise an invertible function as one with a g satisfying gof = identity on X and fog = identity on Y; g is the inverse f-inverse.
  14. Prove a function invertible by showing it is one-one and onto — Use the key result: f is invertible if and only if f is one-one and onto, so bijectivity is the test for invertibility.
