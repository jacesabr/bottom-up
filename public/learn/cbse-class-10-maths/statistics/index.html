Statistics

This chapter of CBSE Class 10 Maths covers 23 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Find mean, mode and median of grouped data — Given a grouped frequency distribution, produce its mean, mode, and median by choosing the right formula for each measure.
  2. Compute the class mark (mid-point) of a class interval — The class mark xᵢ is the mid-point of a class interval: (lower limit + upper limit) / 2.
  3. Compute mean of grouped data by the direct method — Multiply each class mark by its frequency, sum the products, then divide by the total frequency: x̄ = Σfᵢxᵢ / Σfᵢ.
  4. Build the fᵢxᵢ column and its total Σfᵢxᵢ — Multiply each frequency by its class mark row by row, then total both the fᵢxᵢ column and the fᵢ column.
  5. Choose an assumed mean 'a' — Pick a convenient class mark — usually the central one — as the assumed mean a to simplify the arithmetic.
  6. Compute deviations dᵢ = xᵢ − a — Subtract the assumed mean a from every class mark to get signed deviations dᵢ.
  7. Compute mean by the assumed mean method — Use x̄ = a + Σfᵢdᵢ / Σfᵢ where dᵢ = xᵢ − a are the deviations from an assumed mean.
  8. Compute step deviations uᵢ = (xᵢ − a)/h — Divide each deviation dᵢ by the class size h to produce small integer step deviations uᵢ.
  9. Compute mean by the step-deviation method — Use x̄ = a + h × (Σfᵢuᵢ / Σfᵢ) where uᵢ = (xᵢ − a)/h.
  10. Apply step-deviation method when class sizes are unequal — Even with unequal class widths, choose a suitable divisor h and apply x̄ = a + h × (Σfᵢuᵢ / Σfᵢ).
  11. Choose an appropriate method to find the mean — When xᵢ and fᵢ are small use the direct method; when they are large, switch to assumed-mean or step-deviation.
  12. Identify the modal class — Scan the frequency column for the maximum value; the class interval with that maximum frequency is the modal class.
  13. Read off l, h, f₁, f₀, f₂ for the mode formula — Extract the modal class's lower limit l, class size h, modal frequency f₁, and the frequencies immediately before (f₀) and after (f₂) it.
  14. Apply the grouped-data mode formula — Substitute l, f₁, f₀, f₂, h into Mode = l + ((f₁−f₀)/(2f₁−f₀−f₂)) × h.
  15. Build a less-than cumulative frequency column — Starting from the first class, add each successive frequency to the running total to build the cumulative frequency (cf) column.
  16. Recover class frequencies from a cumulative-frequency table — Given a cumulative-frequency table, subtract consecutive cf values to recover each class's frequency.
  17. Identify the median class using n/2 — Compute n/2, then find the class whose cumulative frequency is the smallest value greater than or equal to n/2.
  18. Read off l, cf, f, h for the median formula — From the identified median class, extract its lower limit l, class size h, its own frequency f, and the cumulative frequency cf of the class immediately before it.
  19. Apply the grouped-data median formula — Substitute l, n, cf, f, h into Median = l + ((n/2 − cf)/f) × h.
  20. Convert inclusive (discontinuous) classes to continuous classes — When class intervals like 118–126, 127–135 have a gap, shift each boundary by half the gap (0.5) to make them continuous before applying mode or median formulas.
  21. Find missing frequencies given the median and total — When two frequencies x and y are unknown, use Σf = n (one equation) and the median formula at the given median value (second equation) to solve for both.
  22. Use the empirical relation 3 Median = Mode + 2 Mean — The empirical relationship 3 Median = Mode + 2 Mean lets you estimate one measure when the other two are known.
  23. Select the appropriate measure of central tendency — Choose mean when all values contribute equally, median when extremes distort the picture, or mode when the most frequent value matters.
