Inverse Trigonometric Functions

This chapter of CBSE Class 12 Mathematics covers 17 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Recall sine, cosine, tangent as ratios and the standard-angle values — Recall from Class XI that sin, cos, tan are side ratios in a right triangle, and the exact values at the standard angles 0, pi/6, pi/4, pi/3, pi/2.
  2. Measure angles in radians and convert with pi rad = 180 degrees — Recall from Class XI that an angle can be measured in radians, where a full revolution is 2pi rad and pi rad = 180 degrees.
  3. Recall cosec, sec, cot as reciprocals of sin, cos, tan — Recall from Class XI that cosec x = 1/sin x, sec x = 1/cos x and cot x = 1/tan x = cos x / sin x.
  4. Recall the allied-angle identities sin(pi - x) = sin x, cos(2pi - x) = cos x — Recall from Class XI that allied angles such as pi - x and 2pi - x give related ratios, e.g. sin(pi - x) = sin x and cos(2pi - x) = cos x.
  5. Recall that an inverse exists only for a one-one and onto function — Recall from Chapter 1 that f has an inverse only if it is one-one and onto; trig functions are not, over their natural domains.
  6. Define the inverse g of a bijective function f — For a one-one onto f : X -> Y, define the unique inverse g : Y -> X with domain of g = range of f and range of g = domain of f.
  7. Understand a branch of an inverse trig function — Recognise that each interval over which a trig function is bijective gives a different branch of its inverse.
  8. State the principal value branch of sin-inverse — Take sin-inverse with domain [-1, 1] and principal range [-pi/2, pi/2].
  9. State the principal value branch of cos-inverse — Take cos-inverse with domain [-1, 1] and principal range [0, pi].
  10. State the principal value branch of tan-inverse — Take tan-inverse with domain R and principal range (-pi/2, pi/2).
  11. State the principal value branch of cot-inverse — Take cot-inverse with domain R and principal range (0, pi).
  12. State the principal value branch of cosec-inverse — Take cosec-inverse with domain R - (-1, 1) and principal range [-pi/2, pi/2] minus {0}.
  13. State the principal value branch of sec-inverse — Take sec-inverse with domain R - (-1, 1) and principal range [0, pi] minus {pi/2}.
  14. Distinguish sin-inverse x from (sin x) to the power -1 — Avoid the classic trap: sin-inverse x is an angle, NOT the reciprocal 1/sin x.
  15. Define the principal value of an inverse trig function — Define the principal value as the value of the inverse function that lies in the principal branch range.
  16. Apply sin(sin-inverse x) = x and sin-inverse(sin x) = x on the right domains — Use the cancellation identities, valid only when x lies in the correct domain.
  17. Evaluate principal values of inverse trig expressions — Combine the principal-branch ranges and cancellation identities to evaluate inverse trig expressions correctly.
