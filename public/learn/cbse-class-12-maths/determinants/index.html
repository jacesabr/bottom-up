CBSE Class 12 Mathematics

Determinants

This chapter of CBSE Class 12 Mathematics covers 13 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Define the determinant of a square matrix — Recognise the determinant as a unique number associated with each square matrix, written |A| or det A.
  2. Evaluate a 2 x 2 determinant — Compute a second-order determinant as a11 a22 - a21 a12.
  3. Find the minor of an element — Form the minor M_ij of a_ij by deleting its row and column and taking the resulting determinant.
  4. Find the cofactor of an element — Form the cofactor A_ij = (-1)^(i+j) M_ij by attaching the chequerboard sign to the minor.
  5. Expand a 3 x 3 determinant along a row or column — Evaluate a third-order determinant by cofactor expansion along any row or column.
  6. Find the area of a triangle using a determinant — Use the determinant of vertex coordinates to get triangle area; collinear points give a zero determinant.
  7. Find the adjoint of a square matrix — Form adj A as the transpose of the matrix of cofactors.
  8. Use the identity A(adj A) = (adj A)A = |A| I — Apply the key theorem linking a matrix, its adjoint, and its determinant.
  9. Recognise a singular matrix — Identify a singular matrix as a square matrix with determinant zero.
  10. Recognise a non-singular matrix — Identify a non-singular matrix as a square matrix with non-zero determinant.
  11. Use the product rule |AB| = |A| |B| — Apply the theorem that the determinant of a product is the product of the determinants.
  12. Classify a system as consistent or inconsistent — Call a system consistent if a solution exists, inconsistent if none does.
  13. Invert a matrix via A-inverse = (1/|A|) adj A using non-singularity — Use the criterion 'A is invertible iff A is non-singular' and the formula A-inverse = (1/|A|) adj A.
