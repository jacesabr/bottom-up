Relations and Functions

This chapter of CBSE Class 12 Mathematics covers 16 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Form the Cartesian product A x B — Build A × B as the set of ALL ordered pairs (a, b) with a in A and b in B — the full grid from which any relation is later carved.
  2. Recognise the identity function I_X — Identify the identity function I_X on a set X: the function that sends every element to itself, I_X(x) = x for all x in X.
  3. Define a relation as a subset of A x B — A relation from A to B is any subset of A × B — the mathematical abstraction drops the requirement for a real-world 'link' between the elements.
  4. Recognise the empty relation — Identify the empty relation: the relation containing no pairs, where no element of A is related to any element of A.
  5. Recognise the universal relation — Identify the universal relation: the relation A × A in which every element of A is related to every element of A.
  6. Test a relation for reflexivity — Decide whether a relation is reflexive: every element must be related to itself, (a, a) ∈ R for every a in A.
  7. Test a relation for symmetry — Decide whether a relation is symmetric: whenever (a₁, a₂) is in R, the reverse (a₂, a₁) must also be in R.
  8. Test a relation for transitivity — Decide whether a relation is transitive: if (a1, a2) and (a2, a3) are in R, then (a1, a3) must be in R.
  9. Define and prove an equivalence relation — Recognise an equivalence relation as one that is reflexive, symmetric AND transitive, and prove it by checking all three.
  10. Form equivalence classes and partitions — Understand how an equivalence relation splits a set into disjoint equivalence classes whose union is the whole set.
  11. Test a function for one-one (injective) — Decide whether a function is one-one: distinct inputs give distinct outputs, i.e. f(x1) = f(x2) implies x1 = x2.
  12. Test a function for onto (surjective) — Decide whether a function is onto: every element of the co-domain Y is the image of some element of X.
  13. Recognise a bijective (one-one and onto) function — Identify a bijection as a function that is both one-one and onto.
  14. Compute the composition of two functions — Form the composite gof of f : A -> B and g : B -> C by (gof)(x) = g(f(x)).
  15. Define an invertible function and its inverse — Recognise an invertible function as one with a g satisfying gof = identity on X and fog = identity on Y; g is the inverse f-inverse.
  16. Prove a function invertible by showing it is one-one and onto — Use the key result: f is invertible if and only if f is one-one and onto, so bijectivity is the test for invertibility.
