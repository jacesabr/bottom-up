Continuity and Differentiability

This chapter of CBSE Class 12 Mathematics covers 16 concepts, taught bottom-up. The AI tutor builds each from the ground up and checks you understand it before the next. Here's what you'll work through:

  1. Test continuity of a function at a point — Decide whether f is continuous at c by checking the limit at c equals f(c).
  2. Define a continuous function — Recognise a continuous function as one continuous at every point of its domain.
  3. Combine continuous functions algebraically — Use that sums, differences, products and quotients (denominator nonzero) of continuous functions are continuous.
  4. Use continuity of a composite function — Apply: if g is continuous at c and f is continuous at g(c), then f o g is continuous at c.
  5. Define the derivative of a function — Define f'(c) as the limit of the difference quotient as h tends to 0.
  6. Test differentiability at a point — f is differentiable at c if the difference-quotient limit exists; equivalently both one-sided derivatives agree.
  7. Use that differentiable implies continuous — Apply the corollary: every differentiable function is continuous (but not conversely).
  8. Differentiate composite functions by the chain rule — Use df/dx = (dv/dt)(dt/dx) for f = v o u with t = u(x).
  9. Differentiate implicit functions — Differentiate both sides of an implicit relation w.r.t. x, treating y as a function of x via the chain rule.
  10. Differentiate inverse trigonometric functions — Use the chain rule on inverse trig functions to obtain their derivatives.
  11. Define the exponential function — Recognise the exponential function with base b > 1, with domain R and positive range.
  12. Define the logarithm as the inverse of the exponential — Define log_b a = x to mean b^x = a; the log function is the inverse of the exponential.
  13. Differentiate e^x and log x — Use d/dx(e^x) = e^x and d/dx(log x) = 1/x.
  14. Differentiate using logarithmic differentiation — Take logs of both sides before differentiating, especially for variable-base-variable-exponent forms.
  15. Differentiate functions in parametric form — For x = f(t), y = g(t), find dy/dx = (dy/dt) / (dx/dt).
  16. Find the second order derivative — Differentiate the first derivative again to obtain d2y/dx2, using the full differentiation toolkit.
